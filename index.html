    Campo Minado

    Calculadora Java

    Este projeto é uma calculadora desenvolvida em Java utilizando a biblioteca Swing para a interface gráfica. A calculadora suporta operações matemáticas básicas, como soma, subtração, multiplicação e divisão, além de funcionalidades como inversão de sinal e uso de números decimais.

    Calculadora JavaScript

    Este projeto implementa uma calculadora funcional usando JavaScript. A lógica está encapsulada em um objeto que gerencia o display, trata eventos de clique nos botões e realiza cálculos. O JavaScript permite inserir números e operações, limpar o display, apagar caracteres e calcular expressões matemáticas.

    Sistema de Controle de Estoque

    Este projeto usa Python e Tkinter para criar um sistema de controle de estoque. Ele permite adicionar e remover produtos, registrar entradas e saídas, e gerar relatórios. Os produtos são carregados e salvos em um arquivo de texto. A interface gráfica facilita a interação do usuário com o sistema.

    Calculadora de IMC

    Este script JavaScript calcula o Índice de Massa Corporal (IMC) com base nos dados inseridos pelo usuário. Ao preencher um formulário com peso e altura, o IMC é calculado e exibido na página junto com uma classificação. O código valida os dados inseridos e fornece feedback caso haja erros ou valores inválidos.
